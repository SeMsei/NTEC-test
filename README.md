# NTEC-test
NTEC-test work

# Installation:
* ## Run script to install all dependencies:
    ```
    sudo bash install-dep.sh
    ```
* ## Create build directory:
    ```
    mkdir build && cd build
    ```
* ## Build project
    ```
    cmake ..
    cmake --build .
    ```
# Usage
 ## Run project:
    Run NTEC file to start server
    ```
    cd build
    ./NTEC
    ```
 ## Run test:
    Run Testing file to start server
    ```
    cd build
    ./Testing
    ```
